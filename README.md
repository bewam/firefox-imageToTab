#Image to tab
An addon to open a image in a new tab.

# INSTALL

* build and/or
* download the .xpi and install with firefox.

#BUILD

__requirements__

* You will need [nodejs](https://nodejs.org) + [npm](https://www.npmjs.com/).


__instructions__

* `npm install jpm`
* `jpm xpi`
* install the resulting .xpi in firefox.
